Using with JQuery

YDN-DB is also distributed as JQuery plug-in module. The plugin script, jquery-ydn-db-min-x.x.js, should be included after jquery script in your HTML file.

Usage is the same as YDN-DB module. When using as JQuery plugin, a db field is introduced to JQuery as ready to use database instance.

A basic use case is as follow: 

$.db.setName('db_name_1');

$.db.setItem('x', 'some value')

$.db.getItem('x').done(function(value) {
    console.log('x = ' + value);
});
Database instance is ready, so that you can invoke database methods, like setItem, before opening the database.
