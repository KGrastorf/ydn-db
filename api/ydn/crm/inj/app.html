Class ydn.crm.inj.App

Main application for injecting content script to gmail.

As soon as the app run, this connect with background script, create stable connection channel and check login status and load of the user setting. Finally this will load SugarCRM panels and listen gmail histry changes. These changes are regularily updated back to Gmail content panel and inject right bar of gmail.

Constructor
ydn.crm.inj.App(model, opt_dom, opt_parent)
Create a new content script app.
Methods
getRightBarTable(opt_log_detail)
In gmail message thread page, sniff main contact of the message and suitable element to inject ui.

The method need to be invoked multiple times until suitable ui and contact data is available since Gmail render asynchroniously.

Parameters:
{boolean=} opt_log_detail
Log process detail. (use only during debug)
Parameters:
{HTMLTableElement}
suitable element to inject app UI.
init()
Initialize UI.
Static Methods

Return:
