Authorizations

Authorization allow authenticated user to access resources with or without proxy servers. Most resources are consumed in RESTful format.

Bearer token based authorization

This is Oauth2 access token authorization. Access token can be obtained using client flow or three laps server flow. Â 

Proxy based authorization

Login server itself can serve as authorization proxy server. In this case, web app send request to login server. The server insert requested authorization headers to the request. This allow very fine grain access control than possible with bearer token based authorization.
